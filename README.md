# SpeedTest

1. Click on the SpeedTest link on order to download the file
2. Unzip the files to a folder on your computer
3. Add a task to the system scheduler so it will run SpeedTest_bezeqb.exe every half hour
4. Whenevr you want to see the result, open the folder and run SpeedTest_analysis

# How to add a task
1. Open Windows Explorer
2. Right Click on My computer
3. Select "Manage"
4. Select "Task Scheduler"
5. On the Actions menu select "Create Task"
6. Enter a name "SpeedTest", click "Triggers"
7. Click New, Choose "daily", at the bottom at the "Advance settings" 
8. Select "Repeat task every" and select "30 minutes" from the menu
9. Open the "for a duration of" menu and select "indefinitely", click OK
10. Click Actions, then click New
11. Click Browse, open the folder where you saved the files and choose the file named SpeedTest_bezeQ.exe
12. In "Start in (optional)" enter the name for that folder (copy it from the begining of the file name, click OK
13. Click OK, to save it
14. To test the task, search for it from the bottom window, double clik on the name to oprn it and select Run from the side menu
